Boas Vindas ao Guia BH & Região!

O objetivo do sistema é permitir que usuários encontrem locais, avaliem serviços e gerenciem seus lugares favoritos de forma simples e intuitiva. Este projeto foi desenvolvido como parte do curso de **Análise e Desenvolvimento de Sistemas da PUC Minas**, aplicando conceitos de desenvolvimento Web Front-end e manipulação de dados JSON.

## Funcionalidades Principais

* ✅ **Catálogo de Locais:** Listagem dinâmica de cards com informações turísticas.
* ✅ **Sistema de Busca:** Filtros para encontrar locais específicos.
* ✅ **Autenticação:** Telas de Login e Cadastro de Usuários (com validação básica).
* ✅ **Favoritos (CRUD):** Funcionalidade para o usuário salvar e remover locais da sua lista pessoal.
* ✅ **Melhores Avaliados:** Seção destacando os locais com melhor classificação.
* ✅ **Responsividade:** Interface adaptada para Desktop, Tablets e Celulares.

INSTRUÇÕES PARA EXECUÇÃO:

1. Instale as dependências (se necessário):
   npm install

2. Inicie o Back-end (JSON Server):
    npx json-server ./db.json 

3. Abra o arquivo /public/index.html no navegador para iniciar o front-end.
